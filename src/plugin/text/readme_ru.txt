Observer 1.0.1
Плагин для FAR 1.71+
Автор: Ariman

-------------------------------------------------------------------

1. Общее описание.

Плагин предназначен для просмотра и извлечения файлов
и разнообразных файловых контейнеров.
Операции добавления/удаления/изменения файлов не предусмотрены.

Поддерживаемые форматы:

- MSI пакеты для Windows Installer-а

- Образы UDF (ISO 13346).

- Образы CD/DVD дисков : ISO-9660 (incl. Joliet, RockRidge), CUE/BIN, MDF/MDS

- Инсталляционные пакеты, сделаные программой NSIS

2. Лицензия и авторские права.

Плагин включает в себя код других авторов, с некоторыми изменениями.
Вследствии чего распространияется под LGPL.

Используемый код:

- 7-zip (взята поддержка UDF + NSIS). От него отнаследована лицензия LGPL.
Модуль для UDF взят как есть, только лишнее обрезано.

- Total Commander ISO plugin by Sergey Oblomov.
Передалан на поддержку wchar_t в именах. Базируется на версии 1.7.4 b1.
Лицензии на код не нашел, поэтому тоже пойдет по LGPL.

3. Системные требования.
Минимальная версия FAR-а для работы плагина 1.71, build 2406 (и выше).
Для поддержки MSI нужен установленный Windows Installer, желательно 4+.
OS: Win2k+ (тестилось на WinXP и Win2003, но на 2k должно тоже работать).

4. Гарантии.
Плагин предоставляется по принципу "as-is" без каких-либо гарантий,
явных или неявных. Используйте на свой страх и риск. ;-)
